# 🚀 GrowEasy AI CSV Importer

An AI-powered CSV Importer built using React, FastAPI, and Google Gemini AI.

## Features

- CSV Upload
- CSV Preview
- AI-based CRM Field Mapping
- CRM Record Extraction
- Validation
- Import Summary
- Responsive UI

## Tech Stack

Frontend
- React
- Axios
- CSS

Backend
- FastAPI
- Python
- Pandas
- Google Gemini AI

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn app:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm start
```
